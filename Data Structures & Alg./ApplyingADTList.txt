Write a function Reverse, the function takes in an ADT List, 
returns a new ADT List with the reversed elements.

function Reverse(list L)
    m.List()
    m.initialize()
    for(int i = 0; i <= L.Size; i++)
        m.insert(L.get(L.Size(i)), 0)
        m.Remove(L.Size())

    return m.List()
